To access our game go to: https://nmurthy9.github.io/RGB-Game/

The goal of the RGB game is to guess which color is represented by the R,G,B values given.
Cycle through the colors and choose the correct match!

Controls:
  - a key navigates through the colors
  - d key selects a color
  - r key resets the game

** Tips **
  - Navigating with the 'a' key goes from left to right starting on the top row and moving to the bottom.
  - Numbers on the color buttons will be highlighted when the are navigated through.
